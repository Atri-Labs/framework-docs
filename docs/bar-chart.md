## Custom properties of bar chart

All the custom properties of bar chart must be accessed

### `Bar.custom.data`

```json
at.bar.custom.data = [
    {
        "x": "2010",
        "temperature": 20,
        "rainfall": 10
    },
    {
        "x": "2011",
        "temperature": 30,
        "rainfall": 20
    },
    {
        "x": "2012",
        "temperature": 20,
        "rainfall": 15
    }
]
```
